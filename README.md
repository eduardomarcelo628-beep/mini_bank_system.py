#  💳 Sistema Bancário - Python
Um projeto pessoal criado para praticar alguns conceitos da programação, como condições if/else, lógica de programação e outros assuntos interessantes. 

Este projeto foi criado com o objetivo de colocar em prática conceitos fundamentais da linguagem, simulando operações básicas de uma conta bancária diretamente pelo terminal.

## 🚀 Funcionalidades

- 💰 Depósito de Valores
- 💸 Saque de dinheiro
- 📊 Consulta de saldo
- 🚪 Encerramento do sistema
- ✅ Validação de saldo insuficiente

## 🛠️ Tecnologias Utilizadas

- Python 3

## 📚 Conceitos praticados

Durante o desenvolvimento deste projeto, pude praticar diversos conceitos importantes da programação, como:

- Estruturas de decisão (if, elif, else)
- Estruturas de repetição (while)
- Entrada de dados (input)
- Saída de dados (print)
- Manipulação de variáveis
- Operadores matemáticos
- Lógica de programação

## ▶️ Como executar

1. Clone este repositório:

git clone https://github.com/eduardomarcelo628-beep/mini_bank_system.git

2. Acesse a pasta do projeto:
   
cd bank-system

3. Execute o arquivo Python:

python bank_system.py

## 🎯 Objetivo

Meu objetivo com este projeto foi fortalecer minha lógica de programação e compreender como funcionam operações básicas presentes em sistemas bancários.

Cada projeto desenvolvido representa mais um passo na minha jornada de aprendizado em desenvolvimento de software.
