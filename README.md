# Sistema Bancário - Python
Um projeto pessoal criado para praticar alguns conceitos da programação, como condições if/else, lógica de programação e outros assuntos interessantes. 

Este projeto foi criado com o objetivo de colocar em prática conceitos fundamentais da linguagem, simulando operações básicas de uma conta bancária diretamente pelo terminal.

## Funcionalidades

- 💰 Depósito de Valores
- 💸 Saque de dinheiro
- 📊 Consulta de saldo
- 🚪 Encerramento do sistema
- ✅ Validação de saldo insuficiente

## Tecnologias Utilizadas

- Python 3
